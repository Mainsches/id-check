# AI Identity Risk Scanner

Minimal MVP built with:

- Next.js (App Router)
- Route Handler API (`app/api/scan/route.ts`)
- Vercel-ready deployment
- No login
- No database
- No payment

## Features

- Simple identity input form
- Mock scan endpoint
- Risk score from 0–100
- Risk level: Low / Medium / High
- Findings
- AI-style summary
- Recommendations

## Run locally

```bash
npm install
npm run dev
